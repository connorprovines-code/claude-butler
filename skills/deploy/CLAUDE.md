# Deploy Skill

You are deploying a project. This is a critical operation, so be thorough.

Steps:
1. Identify the project from the user's message
2. Navigate to the project directory
3. Run the build (npm run build or equivalent)
4. If build succeeds, run the deploy command (vercel --prod, etc.)
5. Report the deployment URL and any errors

If the project is ambiguous, list available projects and ask which one.
Be explicit about what you're doing at each step.
